# Задача: Запрос к Базе Знаний (Ask Wiki)

Вы должны ответить на вопрос пользователя, используя исключительно информацию из базы знаний в каталоге `vault/wiki/`.

## Данные запроса:
- **Вопрос пользователя**: {{QUESTION}}

---

## Инструкция по выполнению:

1. **Поиск по Базе Знаний**:
   - Используйте встроенные инструменты поиска (например, `Grep`, `Glob`, `Read`) для нахождения упоминаний ключевых слов из вопроса в файлах `vault/wiki/`.
   - Изучите связанные страницы.

2. **Формирование ответа**:
   - Напишите лаконичный структурированный ответ на русском языке.
   - Опирайтесь только на факты, зафиксированные в базе знаний. Если информации нет, честно укажите это.
   - Ссылайтесь на страницы базы знаний, используя синтаксис `[[Имя Страницы]]`.
   - **ВНИМАНИЕ**: Данная задача предназначена ТОЛЬКО для чтения. Ни в коем случае НЕ создавайте и НЕ изменяйте файлы в `vault/`.

3. **Завершение**:
   - Выведите маркеры результата на новых строках в самом конце вашего финального ответа:
     ```
     RESULT: wiki
     SUMMARY: <Краткое резюме ответа на вопрос в 1-2 предложениях>
     ```
