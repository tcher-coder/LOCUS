# Задача: Импорт Видео — Фаза 1: Конспект (Ingest Video)

Вы должны обработать видеоматериал по указанному URL с помощью скилла `/watch` и создать конспект-выжимку в `raw/` по правилам `vault/AGENTS.md`.

**ВАЖНО: На этой фазе вы НЕ трогаете `wiki/`** (ни страниц, ни log.md, ни index.md). Интеграция в вики будет отдельной задачей после доставки конспекта пользователю.

## Данные запроса:
- **URL видео**: {{URL}}
- **Включить кадры (Vision)**: {{FRAMES_ENABLED}} (True / False)
- **Дополнительные инструкции владельца**: {{INSTRUCTIONS}}

---

## Инструкция по выполнению:

1. **Запуск скилла /watch**:
   - Вызовите скилл `/watch {{URL}}`.
   - **Режим выполнения**:
     - Если `{{FRAMES_ENABLED}}` равен `True` (флаг `+frames`), используйте режим `efficient` (~50 ключевых кадров).
     - Иначе — режим `transcript` (только субтитры/транскрипт, для экономии vision-токенов).
   - При отсутствии субтитров автоматически сработает Whisper-фолбэк (`GROQ_API_KEY` уже в окружении).
   - Если транскрипт получить не удалось (нет субтитров и Whisper вернул ошибку):
     - Прекратите выполнение и выведите в самом конце ответа:
       ```
       RESULT: NONE
       SUMMARY: Ошибка распознавания видео: субтитры недоступны, Whisper-фолбэк завершился ошибкой.
       ```

2. **Создание конспекта-выжимки**:
   - Сформируйте Markdown-документ строго по структуре и правилам стиля из `vault/AGENTS.md` (раздел 2):
     оглавление в начале, TL;DR, ключевые идеи, затем сжатое изложение сути по смысловым разделам (не по хронологии видео).
     Пишите как самостоятельное ёмкое исследование: **без таймкодов, без цитат, без «автор говорит» и отсылок к видео**,
     термины с пояснением в скобках, **без рекламных и спонсорских вставок** (интеграции, промокоды, призывы подписаться — полностью пропускать и не упоминать), без воды. Никаких полотен текста: короткие абзацы по 2–4 предложения через пустую строку, перечисления — маркированными списками. Объём пропорционален источнику (ориентиры в AGENTS.md): ролик до ~10 минут — 1–3 минуты чтения, 10–40 минут — 3–5, длинное видео от ~40 минут — развёрнуто, до 6–12 минут.
     Mermaid-диаграммы запрещены; Markdown-таблицы — можно.
   - Сохраните его в `raw/YYYY-MM-DD-slug.md` (где `slug` — краткое английское название видео латиницей в нижнем регистре через дефис, а `YYYY-MM-DD` — текущая дата).
   - Примените дополнительные инструкции владельца {{INSTRUCTIONS}}, если указаны.

3. **Завершение** (сразу после сохранения файла конспекта, БЕЗ обновления вики):
   - Выведите маркеры результата на новых строках в самом конце вашего финального ответа:
     ```
     RESULT: raw/YYYY-MM-DD-slug.md
     SUMMARY: <TL;DR видео в 1-3 предложениях, ЧИСТЫМ текстом без Markdown-разметки — уйдёт подписью к файлу в канал>
     ```
