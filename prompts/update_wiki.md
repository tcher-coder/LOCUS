# Задача: Фаза 2 — Обновление Вики (Память)

Конспект уже создан и доставлен пользователю: `{{DOC_PATH}}`.
Теперь интегрируйте знания из него в вики строго по правилам `vault/AGENTS.md` (раздел 3).

## Инструкция по выполнению:

1. Перечитайте `{{DOC_PATH}}`, если содержимое не в контексте.
2. Обновите существующие или создайте новые страницы концептов в `wiki/topics/`, `wiki/people/`, `wiki/sources/` (от 5 до 15 страниц):
   - **Не плодите дубликаты** — сначала найдите поиском (Grep/Glob) похожие страницы; существующие темы дополняйте.
   - Страницы вики — только чистый текст (абзацы, списки, цитаты) с перекрёстными ссылками `[[Имя Страницы]]`. Таблицы и Mermaid-диаграммы в вики **запрещены**.
   - Противоречия с уже имеющейся информацией фиксируйте явно, не перезаписывая старое молча.
   - В конце каждой созданной/обновлённой страницы — блок «Источники» со ссылкой на оригинал.
3. Обновите `wiki/log.md` (запись об операции по формату из AGENTS.md).
4. Обновите `wiki/index.md` (новые концепции — в соответствующие разделы навигации).

## Завершение:

Выведите маркеры на новых строках в самом конце финального ответа:
```
RESULT: WIKI_UPDATED
SUMMARY: <кратко: какие страницы созданы, какие обновлены>
```
