# Задача: Импорт Статьи — Фаза 1: Конспект (Ingest Article)

Вы должны загрузить статью по указанному URL и создать конспект-выжимку в `raw/` по правилам `vault/AGENTS.md`.

**ВАЖНО: На этой фазе вы НЕ трогаете `wiki/`** (ни страниц, ни log.md, ни index.md). Интеграция в вики будет отдельной задачей после доставки конспекта пользователю.

## Данные запроса:
- **URL статьи**: {{URL}}
- **Дополнительные инструкции владельца**: {{INSTRUCTIONS}}

---

## Инструкция по выполнению:

1. **Загрузка контента**:
   - Используйте `WebFetch` для получения содержимого по ссылке {{URL}}.
   - Если сайт заблокирован, требует подписку (paywall), выдал ошибку (403, 404) или недоступен:
     - Прекратите выполнение. **Не галлюцинируйте** и не придумывайте контент.
     - Выведите в самом конце ответа:
       ```
       RESULT: NONE
       SUMMARY: Ошибка загрузки статьи: <подробное описание ошибки или статус-кода>
       ```

2. **Сценарий Sitemap / Краулинг нескольких страниц**:
   - Если {{URL}} указывает на карту сайта (Sitemap) или вам поручено обойти карту сайта по определенной теме:
     - Найдите ссылки на страницы и отфильтруйте по теме.
     - Если страниц больше **20** — составьте план обработки первых 20 и завершите с:
       ```
       RESULT: NONE
       SUMMARY: Найдено более 20 страниц для краулинга. Пожалуйста, подтвердите план: <описание плана>
       ```

3. **Создание конспекта-выжимки**:
   - Сформируйте Markdown-документ строго по структуре и правилам стиля из `vault/AGENTS.md` (раздел 2):
     оглавление в начале, TL;DR, ключевые идеи, затем сжатое изложение сути по смысловым разделам.
     Пишите как самостоятельное ёмкое исследование: **без цитат, без «автор пишет» и отсылок к статье**,
     термины с пояснением в скобках, **без рекламных и спонсорских вставок**, без воды. Никаких полотен текста: короткие абзацы по 2–4 предложения через пустую строку, перечисления — маркированными списками. Объём — примерно 1/10 объёма текста статьи, в границах 300–2500 слов (точное правило в AGENTS.md).
     Mermaid-диаграммы запрещены; Markdown-таблицы — можно.
   - Сохраните его в `raw/YYYY-MM-DD-slug.md` (где `slug` — транслитерация названия статьи латиницей в нижнем регистре через дефис, а `YYYY-MM-DD` — текущая дата).
   - Если владелец оставил дополнительные инструкции {{INSTRUCTIONS}}, примените их.

4. **Завершение** (сразу после сохранения файла конспекта, БЕЗ обновления вики):
   - Выведите маркеры результата на новых строках в самом конце вашего финального ответа:
     ```
     RESULT: raw/YYYY-MM-DD-slug.md
     SUMMARY: <TL;DR материала в 1-3 предложениях, ЧИСТЫМ текстом без Markdown-разметки — уйдёт подписью к файлу в канал>
     ```
